Vamos colocar alguns símbolos especiais:

™ = Trade Mark

€ = Euro

£ = Libra (pound)

¥ = Yen

¢ = Cent

∅ = Vazio

∑ = Soma

Δ = Delta

← = Seta esquerda

↑ = Seta acima

→ = Seta direita

↓ = Seta baixo
